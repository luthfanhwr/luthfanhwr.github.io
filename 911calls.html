911 Calls Project

September 2021

For this capstone project we will be analyzing some 911 call data from Kaggle. There will be data visualization and data engineering to gain the most information that we possibly can. This project was done during the course by Jose Portia through Udemy.com.

The data contains the following fields:

  1. lat : String variable, Latitude
  2. lat : String variable, Latitude
  3. lng: String variable, Longitude
  4. desc: String variable, Description of the Emergency Call
  5. zip: String variable, Zipcode
  6. title: String variable, Title
  7. timeStamp: String variable, YYYY-MM-DD HH:MM:SS
  8. twp: String variable, Township
  9. addr: String variable, Address
  10. e: String variable, Dummy variable (always 1)
