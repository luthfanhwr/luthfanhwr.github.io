Bank Stocks Finance Project

September 2021

In this data project we will focus on exploratory data analysis of stock prices. This project is meant to practice visualization and pandas skills, it is not meant to be a robust financial analysis or be taken as financial advice.

We'll focus on bank stocks and see how they progressed throughout the financial crisis in the US all the way to early 2016.

Let's import all the necessary libraries first.




We will get stock information for the following banks from the pickle file.

  1. Bank of America
  2. CitiGroup
  3. Goldman Sachs
  4. JPMorgan Chase
  5. Morgan Stanley
  6. Wells Fargo




We will now start the data exploration process. Extract the maximum Closing price for the stocks.
We will use the .xs() method to do this Multi-Level Indexing.




Create a new empty DataFrame called returns. This dataframe will contain the returns for each bank's stock.




We can use pandas pct_change() method on the Close column to create a column representing this return value. Create a for loop that goes and for each Bank Stock Ticker creates this returns column and set's it as a column in the returns DataFrame




Create a pairplot using seaborn of the returns dataframe. There should be one plot that stands out.




We can see that CitiGroup has a bit strange behavior on the plot.
This is due to stock crash that hit Citigroup. Background on Citigroup's Stock Crash available here. We will also see the enormous crash in value if you take a look a the stock price plot (which we do later in the visualizations.)

Using this returns DataFrame, figure out on what dates each bank stock had the best and worst single day returns. We should notice that 4 of the banks share the same day for the worst drop due to some reason.




You should have noticed that Citigroup's largest drop and biggest gain were very close to one another. This is due to Citigroup stock split.




Take a look at the standard deviation of the returns and determine which stock could be classified as the riskiest over the entire time period.




Also classify the riskiest stock during year 2015.




Create a distplot using seaborn of the 2015 returns for Morgan Stanley.




Create a distplot using seaborn of the 2008 returns for CitiGroup.

Create a line plot showing Close price for each bank for the entire index of time.

Let's analyze the moving averages for these stocks in the year 2008.

Plot the rolling 30 day average against the Close Price for Bank Of America's stock for the year 2008




Create a heatmap of the correlation between the stocks Close Price.




Use seaborn's clustermap to cluster the correlations together.




Use .iplot(kind='candle) to create a candle plot of Bank of America's stock from Jan 1st 2015 to Jan 1st 2016.




Use .ta_plot(study='sma') to create a Simple Moving Averages plot of Morgan Stanley for the year 2015.




Use .ta_plot(study='boll') to create a Bollinger Band Plot for Bank of America for the year 2015.




We have now finished analyzing the Bank Stock Data and have gained a few insights about what this data tells us and might possibly be useful to determine which Stock might be the right fit to invest in.
